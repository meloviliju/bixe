Fit to the double-power law

    The double-power law model (as illustrated in https://arxiv.org/abs/1212.1362) predicts the probability that a word with rank r occurs as follows:
        p(r) = C * r^{-α}               for r <= b
             = C * b^{γ-α} * r^{-γ}      for r > b
    with the normalization constant
        C = 1 / ( Σ_{r=1}^{b} r^{-α} + b^{γ-α} Σ_{r=b+1}^{N} r^{-γ} ).
